PROMPT: Let B and N be cardinal direction relations. If region x is B of region y and region y is N of region z, then which could the possible relations between region x and region z be?

ANSWER:

From the definitions, the two given assumptions are:

1) x B y  
   ⇔  infx(y) ≤ infx(x),  supx(x) ≤ supx(y),  infy(y) ≤ infy(x),  supy(x) ≤ supy(y).

2) y N z  
   ⇔  supy(z) ≤ infy(y),  infx(z) ≤ infx(y),  supx(y) ≤ supx(z).

We want the possible relation(s) between x and z.

────────────────────────────────────────────────────────
1) Vertical (y–coordinate) constraints
────────────────────────────────────────────────────────

•  From x B y, we have  
       infy(y) ≤ infy(x)   and   supy(x) ≤ supy(y).

•  From y N z, we have  
       supy(z) ≤ infy(y).

Putting these together:  
   supy(z) ≤ infy(y) ≤ infy(x).  

Hence the top of z is at or below the bottom of x, so on the vertical axis x is necessarily north of z:

   supy(z) ≤ infy(x).

────────────────────────────────────────────────────────
2) Horizontal (x–coordinate) constraints
────────────────────────────────────────────────────────

•  From x B y,  
       infx(y) ≤ infx(x)   and   supx(x) ≤ supx(y).

•  From y N z,  
       infx(z) ≤ infx(y)   and   supx(y) ≤ supx(z).

Together these give the “sandwich”:

   infx(z) ≤ infx(y) ≤ infx(x) ≤ supx(x) ≤ supx(y) ≤ supx(z).

Hence in the x–direction, z’s bounding box covers (or is at least as wide as) y’s, and y’s covers x’s; in particular,

   infx(z) ≤ infx(x)
   and
   supx(x) ≤ supx(z).

────────────────────────────────────────────────────────
3) Checking the definition of x N z
────────────────────────────────────────────────────────

Recall x N z means:  
   supy(z) ≤ infy(x),    infx(z) ≤ infx(x),    supx(x) ≤ supx(z).

We have just seen:

•  supy(z) ≤ infy(x)   (vertical constraint).  
•  infx(z) ≤ infx(x),  supx(x) ≤ supx(z)   (horizontal constraints).

So indeed all three hold, showing x N z.

────────────────────────────────────────────────────────
Conclusion
────────────────────────────────────────────────────────

No other cardinal‐direction configuration can be satisfied simultaneously with x B y and y N z (one sees contradictions in trying to fulfill the requirements for E, W, S, etc.). Hence the only possible relation is:

x N z.
